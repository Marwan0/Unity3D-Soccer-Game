# Unity3D-Soccer-Game

Open the project with Unity3D 5.4

## iOS build
 - add to Xcode the following frameworks (needed by Heyzap/Chartboost/Admob)
 - safari
 - eventkit
 - eventui
 - mobilecoreservices
 - messageui
 - corebluetooth
 - coretelephony

## ADs
- Chartboost	v6.4.4
- HeyZap		v9.5.7
- Admob		v7.8.1


- Goalkeeper and Forward have really efficient AI.
